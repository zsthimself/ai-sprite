# App 目录

包含页面和布局的 Next.js App Router 目录。

- `[layout.tsx]`: [根布局 - 定义全局样式、字体和元数据]
- `[page.tsx]`: [路由入口 - 编排 GeneratorForm、SpritePreview 和 UsageBanner]
- `[globals.css]`: [全局样式 - Tailwind 指令和基础样式]
- `[favicon.ico]`: [资源 - 应用程序图标]
- `[privacy/page.tsx]`: [法律页面 - 隐私政策]
- `[terms/page.tsx]`: [法律页面 - 服务条款]
